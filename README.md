# SmartCart Clustering System

## Overview

SmartCart is a machine learning-based customer segmentation system designed to analyze shopping behavior and purchasing patterns. The project uses clustering algorithms to group customers based on similar buying habits, helping businesses improve targeted marketing and customer engagement strategies.


## Features

* Customer segmentation using machine learning
* Shopping behavior and spending pattern analysis
* K-Means and Agglomerative Clustering implementation
* Data preprocessing and feature scaling
* Data visualization for customer insights
* Comparative analysis of clustering techniques


## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn


## Machine Learning Algorithms

### K-Means Clustering

Used to group customers into clusters based on purchasing behavior and spending patterns.

### Agglomerative Clustering

Implemented hierarchical clustering to compare clustering accuracy and customer grouping efficiency.


## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Feature Selection and Scaling
4. Exploratory Data Analysis (EDA)
5. Customer Segmentation using Clustering Algorithms
6. Visualization of Customer Groups
7. Performance Comparison and Analysis


## Results

* Successfully identified customer segments with similar purchasing behavior
* Improved clustering analysis by comparing multiple algorithms
* Generated visual insights to support business decision-making and personalized marketing strategies
